
Contact Manager 
Description
A React app to manage contacts by adding and displaying their details like name, email, and phone number.

Features
- Add new contacts
- View contact list with timestamps

Technologies Used
- React 17
- react-simple-code-editor (for code editing)
- prismjs (for syntax highlighting in the editor)

How to Run
1. Clone the repo  
2. Run `npm install`  
3. Run `npm start`
